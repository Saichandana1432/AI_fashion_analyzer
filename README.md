# AI Fashion Analyzer & Metadata Generator

An interdisciplinary Computer Vision project bridging AI Engineering with Fashion Technology. This system automatically processes apparel imagery or design sketches to classify garment categories and extract dominant color metadata.

## 🚀 Live Demo (Google Colab)
* Developed and tested using Google Colab utilizing a Vision Transformer (ViT) pipeline.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Deep Learning Framework:** PyTorch / Hugging Face Transformers
* **Machine Learning:** Scikit-Learn (K-Means Clustering)
* **Image Processing:** Pillow (PIL), NumPy

## 📊 Sample Output (Tested on Hand-Drawn Gown Sketch)
* **Detected Garment:** DRESS
* **AI Confidence Score:** 73.58%
* **Extracted Hex Color:** #AF7F7A
